## 🧠 Overview

This project is a **modern AI chat frontend** built with React and Tailwind CSS, designed to interact seamlessly with a FastAPI backend.

It delivers a **smooth, real-time conversational experience** with support for:
- Streaming AI responses (token-by-token)
- File uploads for document-based queries
- Clean and responsive UI

> Think of it as a **mini ChatGPT-style interface — fully customizable and developer-friendly.**

---

## ✨ Features

### 💬 Interactive Chat
- Clean and responsive chat UI
- Message history with structured rendering

### ⚡ Real-Time Streaming
- Server-Sent Events (SSE)
- Token-by-token response rendering

### 📄 File Upload
- Upload PDFs for AI processing
- Designed for RAG pipelines

### 🎨 Modern UI
- Tailwind CSS v4
- Smooth animations and layout

### 🔗 Backend Integration
- Axios-powered API communication
- Plug-and-play with FastAPI
